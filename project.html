    어플리케이션·아두이노 기반 무인 계산 시스템



    대학교 2학년 때 프로젝트 경험을 쌓고자 학부융합 프로그램이라는 팀 프로젝트에 참여하였습니다.
    주제를 '어플리케이션·아두이노 기반 무인 계산 시스템'로 선정하여 어플리케이션 개발 역할을 맡아 진행되었습니다.
    그 당시 안드로이드에 대한 지식이 거의 없어 공부를 병행하며 개발해야 했으나 시간이 촉박하여 개발 완성을 중심으로 진행되었습니다.
    그러나 DB 서버를 여는 것부터 json파일을 만들고 PHP를 사용하여 DB에 접근하는 것까지
    생각보다 안드로이드를 만드는데 필요한 요소들이 너무 많았습니다.

    제가 개발한 화면 중 검색 기능 화면은 검색 텍스트를 입력하고 버튼을 누르면
    해당 텍스트가 들어간 상품명, 브랜드 명을 DB에서 찾고 json파일로 받아와 안드로이드에서 출력하는 거였습니다.
    이를 구현하기 위해서는 PHP를 사용하여 DB에 접근해야 했습니다.
    PHP의 개념이 생소하여 난감했지만 여기저기 자료를 찾아본 결과 DB값을 가져올 수 있게 되었습니다.
    PHP 해결 후 안드로이드에 집중하여 다른 화면 개발을 진행하던 중 다른 팀원의 기능에서도 PHP 파일이 필요해 제게 도움을 요청했었습니다.
    하지만 개발을 중심으로 진행한 나머지 PHP의 개념에 대해 바로 잊어버렸던 상태였습니다.
    결국 큰 도움은 주지 못했습니다. 이때 개발 경험도 중요하겠지만 지식 습득을 놓치고 있었다는 걸 깨달았습니다.
    개발 활동을 하더라도 개념 복습이 중요하다는 걸 깨달았고 블로그에 이에 대한 개념을 정리하게 되었습니다.
    실무에서도 개발에서 그치지 않고 스스로 복습하며 공부하는 자세를 갖고자 노력할 것입니다.
